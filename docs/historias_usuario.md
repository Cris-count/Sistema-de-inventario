# Historias de usuario (RF integrados)

Formato: *Como [rol], quiero [acción], para [beneficio].*

| ID | Historia | Prioridad |
|----|----------|-----------|
| RF-01 | Como usuario, quiero autenticarme con credenciales para acceder según mi rol. | Alta |
| RF-02 | Como Administrador, quiero gestionar usuarios y roles para controlar el acceso. | Alta |
| RF-03 | Como Administrador, quiero gestionar categorías para clasificar productos. | Media |
| RF-04 | Como Administrador, quiero gestionar productos con código único para identificar ítems sin ambigüedad. | Alta |
| RF-05 | Como Administrador, quiero gestionar bodegas para reflejar ubicaciones de stock. | Alta |
| RF-06 | Como Administrador, quiero gestionar proveedores para asociar compras. | Media |
| RF-07 | Como Administrador, quiero registrar stock inicial para fijar la línea base de existencias. | Alta |
| RF-08 | Como Auxiliar/Compras/Admin, quiero registrar entradas para aumentar existencias con trazabilidad. | Alta |
| RF-09 | Como Auxiliar/Admin, quiero registrar salidas validando stock para reflejar egresos reales. | Alta |
| RF-10 | Como Auxiliar/Admin, quiero registrar transferencias entre bodegas sin perder trazabilidad. | Alta |
| RF-11 | Como usuario autorizado, quiero consultar existencias para operar y planificar. | Alta |
| RF-12 | Como Compras u operación, quiero ver alertas de stock mínimo para anticipar faltantes. | Media |
| RF-13 | Como Gerencia u otros roles autorizados, quiero reportes de movimientos para auditoría y análisis. | Alta |
| RF-14 | Como Auxiliar/Admin, quiero registrar conteo físico y ajustes para alinear físico y sistema. | Media |
| RF-15 | Como Administrador, quiero consultar eventos de auditoría de seguridad (si se implementa log). | Baja |
| RF-16 | Como usuario autorizado, quiero exportar reportes (CSV) para análisis externo. | Baja |
